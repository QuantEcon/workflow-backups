## Recent Releases

See [CHANGELOG.md](../../CHANGELOG.md) for detailed changes.

### v0.3.0 (2025-12-02)
- GitHub issues metadata backup (disabled by default)
- Monthly issue reporting with collapsed details
- End-of-month review reminders

### v0.2.0 (2025-12-02)
- Exclusion features (`exclude_archived`, `exclude_repositories`, `exclude_patterns`)
- Improved logging for excluded repos
- Type hints updated to Python 3.9+ style

### v0.1.0 (2025-12-02)
- Repository backup to AWS S3
- Pattern-based repository selection
- Backup verification
- GitHub Actions integration
- OIDC authentication support
